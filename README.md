# README

Dependências: 
- Ruby 2.5.1
- Rails 5.2.1
- Mongodb

Para rodar o projeto:

- Faça o clone do projeto e entre na pasta do mesmo.
- Inicie o serviço do mongodb.
- Instale as gems com bundle install.
- Execute o projeto com rails server.
